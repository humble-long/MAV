# 6 结论

针对仿生扑翼飞行器概念设计阶段领域知识高度分散、设计经验难以复用以及任务驱动方案生成缺乏系统化方法的问题，本文设计并实现了一种知识图谱增强的仿生飞行器设计问答智能体 BioBridge-GraphRAG，以知识图谱作为结构化记忆，并使大语言模型作为智能体在图谱上进行多跳推理与物理校验，并在自构建的 FWMAV-QA 测评数据集上对其性能进行了实证检验。

主要贡献体现在四个方面。第一，在领域知识表达层面，构建了由生物原型层与工程样机层组成的双层知识图谱，并定义了气动相似、运动学相似、形态相似、尺度相似 4 类细分的仿生映射关系，使得"借鉴自哪种生物"既具有定性的语义关联，又具备定量的相似度分数，为生物—工程跨域查询与分析提供了统一的本体支撑。第二，在推理范式层面，提出了尺度律工具增强的图路径推理框架，将 4 个仿生扑翼飞行器领域的物理工具（重量模型、尺度律预测、Strouhal 数与雷诺数判别）以 Function Calling 协议封装为大语言模型可调用的接口，使得语义检索与数值校验能够在同一推理链上协同；与现有通用 GraphRAG 方法相比，该范式明确将"工程参数验证"作为推理的一阶动作。第三，在方案推荐层面，引入基于 3 阶张量 CP 分解的粗筛机制，结合 raw-CP 混合相似度（$\alpha = 0.4$）实现毫秒级 Top-$K$ 召回，并与图路径推理协同构成"粗筛—精排"两阶段范式，从系统层面解耦了语义匹配与物理可行两个判据。第四，在评测资源层面，构建并公开了 FWMAV-QA Benchmark——据作者所知首个面向仿生扑翼飞行器领域的中文知识问答与方案推荐测评数据集（共 754 题、6 类）——以及完整的代码仓库（GitHub + Zenodo DOI）。

本文工作仍存在若干局限。受公开样机数量限制，当前知识图谱仅含 39 架 FlappingWingVehicle，张量分解的潜在因子在小样本下存在一定旋转模糊；与 6 个代表性基线在 FWMAV-QA 全集上的系统对比尚在进行中，§5.5 节将报告更全面的主实验结果与统计显著性检验；案例研究中关于"BioBridge-GraphRAG 优于纯 LLM 直答"的论断目前主要基于定性观察，需在全集评测中给出量化对比。此外，本文的人工评估目前仅由作者本人完成，缺乏多名领域专家的独立校验。

未来工作可从三方面展开：一是将本文方法的 ReAct 单路径推理扩展为 ToG 风格的束搜索，并在并发多候选场景下与本文范式进行权衡；二是随着样机数据规模扩大，进一步引入独立的"组件—性能"维度形成 4 阶张量，并探索贝叶斯张量分解或正则化变体以进一步提升小样本稳健性；三是将双层本体的"生物—工程"映射扩展至材料、加工、控制等其他设计要素，使 BioBridge-GraphRAG 演化为覆盖更完整概念设计链的领域智能助手。
